# Web-based Code Editor

This is a simple web-based code editor that allows you to write and execute JavaScript code in your browser. It features a side-by-side layout with a code editor on one side and an output panel on the other.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (which includes npm)

## Installation

1. Clone the repository or download the source code.
2. Open a terminal in the project directory.
3. Run the following command to install the dependencies:
   ```bash
   npm install
   ```
## Running the Application

To start the server, run the following command:

```bash
npm start
```

Open your web browser and navigate to `http://localhost:3000` to use the application.
